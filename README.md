## Project Overview

This About Me website is the final culmination of my Web Page Development I course. Throughout the semester, I learned how to build clean, semantic HTML structures and organize content using headings, paragraphs, lists, images, and links. As the project progressed, I developed stronger skills in typography, spacing, and color theory, and I learned how to use classes and IDs to structure and style content in a maintainable way.

I also learned how to create responsive layouts using Flexbox, CSS Grid, and media queries so that the site remains readable and visually consistent on mobile, tablet, and desktop devices. In the later stages of the project, I implemented embedded media, a responsive navigation bar, a custom logo, and a favicon. The final version includes a fully written biography and hobbies section, a cleaned and styled image gallery, and a researched blog article on CSS sprites with visual examples. This project reflects my growth as a designer and developer and serves as a foundation for future portfolio work. I look forward to continuing to improve my skills and create even more advanced projects in the future.

## Navigation Bar

The navigation bar is fully responsive and adapts to different screen sizes. On larger screens, the logo and navigation links are displayed in a horizontal layout. On smaller screens (under 60rem), the layout switches to a stacked vertical menu and introduces a hamburger icon for mobile navigation.

The hamburger menu is implemented using the CSS checkbox technique: a hidden checkbox controls the visibility of the menu, and three styled `<span>` elements form the hamburger icon. The navigation links anchor to the main sections of the page (Bio, Gallery, Blog, and Contact), and smooth scrolling is enabled for a polished user experience. This approach keeps the navbar accessible, lightweight, and consistent across devices.

## Color Scheme

The site uses a cohesive color palette centered around pink and burgundy tones.

| Role                | Color Name   | Hex       | RGB                |
|---------------------|-------------|-----------|--------------------|
| Background accent   | Cool purple | #AF7290B2 | rgb(175, 114, 144) |
| Primary accent      | Pink        | #E51177   | rgb(229, 17, 119)  |
| Primary accent dark | Dark pink   | #DD0F73   | rgb(221, 15, 115)  |
| Main text / footer  | Burgundy    | #55062C   | rgb(85, 6, 44)     |

My main text color (burgundy) and background color (cool purple) were checked with a contrast checker and meet the WCAG 4.5:1 contrast ratio requirement for readability.

## Blog Topic – CSS Sprites

My blog article focuses on **CSS sprites**. I explain what sprites are, how they combine multiple small images into a single file, and how this technique reduces HTTP requests and improves page performance. The article also covers how to position individual graphics using the `background-position` property and how sprites can be used for icons, hover states, and interface elements. I included visual examples and code references to demonstrate how sprites can be implemented in real projects.
